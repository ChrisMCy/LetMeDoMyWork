# LetMeDoMyWork - UI/UX Design Specifications

## Design System

### Color Palette (Blue-Gray Professional)
```
Primary Colors:
  - Primary Dark:    #2C3E50  (Dark Blue-Gray) - Headers, Important Text
  - Primary:         #34495E  (Medium Blue-Gray) - Backgrounds
  - Primary Light:   #7F8C8D  (Light Blue-Gray) - Secondary Text
  
Accent Colors:
  - Accent:          #3498DB  (Bright Blue) - Buttons, Links, Active States
  - Accent Dark:     #2980B9  (Darker Blue) - Button Hover/Press
  
Background Colors:
  - Background:      #ECF0F1  (Light Gray) - Main Background
  - Surface:         #FFFFFF  (White) - Cards, Inputs
  - Paused BG:       #E8E8E8  (Light Gray) - Paused TODOs
  
Status Colors:
  - Success:         #27AE60  (Green) - Success Messages
  - Error:           #E74C3C  (Red) - Errors, Max Sends
  - Warning:         #F39C12  (Orange) - Warnings
  - Info:            #3498DB  (Blue) - Info Messages
  
Text Colors:
  - Primary Text:    #2C3E50  (Dark Blue-Gray)
  - Secondary Text:  #7F8C8D  (Light Blue-Gray)
  - Disabled Text:   #BDC3C7  (Very Light Gray)
  - White Text:      #FFFFFF  (On Dark Backgrounds)

Send Count Gradient (White â†’ Red):
  - 0 sends:         #FFFFFF  (White)
  - 25% progress:    #FFB3B3  (Light Pink)
  - 50% progress:    #FF6666  (Medium Red)
  - 75% progress:    #FF3333  (Dark Red)
  - 100% (max):      #E74C3C  (Full Red)
```

### Typography
```
Font Family: 
  - Primary: "Roboto" (Android Default)
  - Fallback: System Default

Font Sizes:
  - Headline 1:      24sp (Bold) - Screen Titles
  - Headline 2:      20sp (Bold) - Section Headers
  - Headline 3:      18sp (Semi-Bold) - Card Titles
  - Body 1:          16sp (Regular) - Primary Text
  - Body 2:          14sp (Regular) - Secondary Text
  - Caption:         12sp (Regular) - Timestamps, Labels
  - Button:          16sp (Medium) - Button Text

Line Heights:
  - Headlines:       1.2x font size
  - Body:            1.5x font size
  - Captions:        1.3x font size
```

### Spacing System (8dp Grid)
```
xs:  4dp   - Minimal spacing
s:   8dp   - Small spacing (between related elements)
m:   16dp  - Medium spacing (between sections)
l:   24dp  - Large spacing (screen padding)
xl:  32dp  - Extra large spacing (major sections)
xxl: 48dp  - Screen top padding
```

### Border Radius
```
Small:     4dp  - Small buttons, chips
Medium:    8dp  - Cards, inputs
Large:     12dp - Bottom sheets, dialogs
Round:     50%  - Circular buttons, avatars
```

### Shadows (Elevation)
```
Level 1:  0dp 1dp 3dp rgba(0,0,0,0.12)  - Subtle (inputs)
Level 2:  0dp 2dp 6dp rgba(0,0,0,0.16)  - Cards
Level 3:  0dp 4dp 12dp rgba(0,0,0,0.20) - Floating buttons
Level 4:  0dp 8dp 24dp rgba(0,0,0,0.24) - Dialogs
```

## Responsive Design

### Screen Sizes
```
Small (< 360dp width):
- Single column layout
- Smaller text (scale 0.9)
- Reduced padding

Medium (360-600dp width):
- Standard layout (as designed)
- Normal text sizes
- Standard padding

Large (> 600dp width / Tablets):
- Two-column on landscape
- Larger cards (max 600dp width, centered)
- Increased padding (24dp instead of 16dp)
```

### Orientation
```
Portrait (default):
- Standard single-column layout
- Full-width cards

Landscape:
- Tablets: Two-column layout (List + Detail)
- Phones: Same as portrait (avoid complexity)
```
