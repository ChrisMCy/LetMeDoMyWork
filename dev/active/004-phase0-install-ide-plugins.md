# Task 004: Install IDE Plugins

## Phase
0 - Project Setup

## Description
Install Flutter and Dart plugins for Android Studio and/or VS Code.

## Steps

### Android Studio:
1. Open Android Studio
2. File â†’ Settings (Windows/Linux) or Preferences (macOS)
3. Navigate to Plugins
4. Search and install "Flutter" plugin
5. Search and install "Dart" plugin (if not auto-installed)
6. Restart Android Studio

### VS Code (Optional):
1. Open VS Code
2. Open Extensions (Ctrl+Shift+X)
3. Search and install "Flutter" by Dart Code
4. Search and install "Dart" by Dart Code
5. Optional: Install "Error Lens", "GitLens"
